# Melodia

**Melodia** is a small school project built with **Next.js**.
It is a simple music browsing application that uses the **iTunes Search API** to search for songs, view album details, and play a preview (no sound though...) through a global player bar.

## Features

- Search for songs using the iTunes API
- View album details and track list
- Global player bar with play/pause controls
- Favorites page (not implemented yet)

## How to Run

1. Install dependencies

```bash
npm install
```

2. Start the development server

```bash
npm run dev
```

3. Open the app in your browser

```
http://localhost:3000
```

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- CSS Modules and Tailwind CSS
- iTunes Search API
